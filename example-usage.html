üìã How to Add Additional Buttons

Simply define a const attConfig = [] array before loading the attribution script. Each button should have text and url properties:

<script>
// Define additional buttons
const attConfig = [
    {
        text: "GitHub Profile",
        url: "https://github.com/username"
    },
    {
        text: "LinkedIn",
        url: "https://linkedin.com/in/username"
    },
    {
        text: "Portfolio",
        url: "https://portfolio.example.com"
    }
];
</script>

<!-- Load the attribution script -->
<script src="build/attribution.min.js"></script>
